# Java语言设计模式

Java语言设计模式总结和Demo，部分参考《Head First Design Patterns》。



设计模式是一套被反复使用的代码设计经验的总结

目的：使用设计模式是为了可重用代码、让代码更容易倍他人理解、保证代码可靠性
